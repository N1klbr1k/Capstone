About this Website

Thanks for visiting my Website! On the main page you can manage creatures in dnd 5e. On the add monster page you can choose to add a monster to the database, and on this page I tell you all about this website.

In order to run a pack, simply add the creatures you want using the dropdown in the left hand column, and press attack. It's that simple. Note that this will always make an attack for every creature in your chosen pack. if you wish to run only one attack, just make sure there is only one creature in the lefthand column.
